# AutoPrep

An AI-powered study assistant that helps students **clear doubts, generate practice tests, and learn interactively** using agent-based architectures built with LlamaIndex.

---

##  Overview

AutoPrep is designed to simulate a **personal AI tutor** that assists students in real-time.

It enables:
-  Concept understanding via conversational AI
-  Automated test generation
-  Instant doubt resolution
-  Adaptive learning experience

Built using **LlamaIndex agents**, AutoPrep leverages structured reasoning and retrieval to deliver high-quality educational assistance.

---

##  Features

-  AI-powered doubt solving (chat-based interface)
-  Dynamic test generation based on topics
-  Personalized learning experience
-  WhatsApp-style chat UI for intuitive interaction
-  Authentication (login/signup flows)
-  Dashboard for tracking usage (future scope)

---

##  Tech Stack

- **Frontend**: React + TypeScript (Next.js)
- **Backend**: API routes (Next.js / Node)
- **AI Framework**: LlamaIndex (Agent-based architecture)
- **Styling**: CSS
- **Authentication**: Custom auth flows

---

##  How It Works

1. User interacts via chat interface
2. Query is processed by LlamaIndex agent
3. Agent retrieves relevant knowledge / generates response
4. System adapts output based on:
   - Topic
   - Difficulty
   - Context
5. Output is returned in conversational format
